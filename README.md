# **Console Chat**

Многопользовательский консольный чат на C# с клиент-серверной архитектурой.

## Архитектура проекта:

- **Chat.Server** — TCP сервер, принимает подключения клиентов и обрабатывает сообщения.
- **Chat.Client** — консольный клиент для подключения к серверу и отправки сообщений.
- **Chat.Shared** — общие модели и контракты, используемые сервером и клиентом.

## Технологии:

- .NET
- TCP sockets
- C#
